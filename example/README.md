# Example for nebula-spark-connector

## How to compile

for spark2.2:
```agsl
cd example
mvn clean package -Dmaven.test.skip=true -Pspark2.2
```

for spark2.4:
```agsl
cd example
mvn clean package -Dmaven.test.skip=true -Pspark2.4
```

for spark3.x
```agsl
cd example
mvn clean package -Dmaven.test.skip=true -Pspark3.0
```
